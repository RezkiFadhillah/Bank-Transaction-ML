# Bank Transaction Clustering & Classification 🚀

Proyek ini menggabungkan Unsupervised Learning (Clustering) dan Supervised Learning (Classification) untuk menganalisis transaksi bank.

## 📁 Struktur Folder
- `notebooks/`: Proses analisis (EDA, Clustering, & Classification).
- `data/`: Dataset hasil segmentasi.
- `models/`: Model machine learning (.h5).
